# Claude PHP & Laravel Kit Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Ship a GitHub repo that curates PHP and Laravel rules, skills, subagents, slash commands, and hooks for Claude Code — distributed as copy-paste files with a `paste-into:` frontmatter contract and CI-enforced index.

**Architecture:** Two top-level stacks (`php/`, `laravel/`) each containing `rules/`, `skills/`, `agents/`, `commands/`, `hooks/`. A PHP script walks the tree, validates frontmatter, and regenerates the README contents index. GitHub Actions runs both checks on every PR. No runtime, no installer — just files.

**Tech Stack:** Markdown, YAML frontmatter, PHP 8.2 (for validator and index generator scripts — thematically appropriate, and PHP is already available in GitHub Actions via `shivammathur/setup-php`), GitHub Actions.

**Design doc:** `docs/plans/2026-04-13-claude-php-laravel-kit-design.md`

**Working directory:** `/var/www/html/claude-php-laravel-kit`

## Conventions used throughout this plan

- **Every task ends in a commit.** Small, focused commits over big mega-commits.
- **Script tasks follow TDD** — write the failing test (fixture-based), watch it fail, implement, watch it pass, commit.
- **Content tasks are batched** per folder because each file is a short markdown document with a fixed frontmatter shape. The "test" for content files is `scripts/lint-frontmatter.php` and `scripts/generate-index.php --check` at the end of each batch.
- **Curation sourcing:** primary source is `affaan-m/everything-claude-code` on GitHub. For each curated file, fetch the upstream file via `gh api` or `curl`, adapt it, and record the source in the file's frontmatter plus `ATTRIBUTIONS.md`. If the upstream license is not MIT-compatible, skip the file and note it.
- **Commit message style:** conventional commits (`feat:`, `docs:`, `chore:`, `ci:`).

## Task 5: Frontmatter validator — implementation

**Files:**
- Create: `scripts/lint-frontmatter.php`

**Step 1: Write the validator**

The script takes one CLI argument (a file or directory path), walks it recursively, and for each `.md` file:

1. Reads the file.
2. Extracts YAML frontmatter between the first `---` and the next `---`.
3. Parses the frontmatter with PHP's built-in parser (use `yaml_parse` if the `yaml` extension is installed; otherwise hand-roll a minimal line-based key: value parser — the frontmatter shape is intentionally flat, so no nested YAML or arrays other than comma-separated lists).
4. Asserts required fields: `name`, `description`, `paste-into`, `stack`, `type`.
5. Asserts either `source` (+ `license`) or `author` is present.
6. Asserts `stack` is one of `php`, `laravel`.
7. Asserts `type` is one of `rule`, `skill`, `agent`, `command`, `hook`.
8. For `type: skill`, asserts the file is named `SKILL.md`.

On any failure, prints `ERROR: <file>: <reason>` to stderr and continues walking. Exits 0 if no errors, 1 if any.

Full implementation (hand-rolled YAML parser — deliberate, avoids the `yaml` extension dependency in CI):

```php
<?php
// scripts/lint-frontmatter.php
declare(strict_types=1);

const REQUIRED_FIELDS = ['name', 'description', 'paste-into', 'stack', 'type'];
const ALLOWED_STACKS = ['php', 'laravel'];
const ALLOWED_TYPES = ['rule', 'skill', 'agent', 'command', 'hook'];

function parseFrontmatter(string $content): ?array
{
    if (!str_starts_with($content, "---\n")) {
        return null;
    }
    $end = strpos($content, "\n---", 4);
    if ($end === false) {
        return null;
    }
    $block = substr($content, 4, $end - 4);
    $out = [];
    foreach (explode("\n", $block) as $line) {
        if (!preg_match('/^([\w\-]+):\s*(.*)$/', $line, $m)) {
            continue;
        }
        $out[$m[1]] = trim($m[2], " \"'");
    }
    return $out;
}

function validateFile(string $path): array
{
    $errors = [];
    $content = file_get_contents($path);
    if ($content === false) {
        return ["$path: could not read file"];
    }
    $fm = parseFrontmatter($content);
    if ($fm === null) {
        return ["$path: missing or malformed frontmatter"];
    }
    foreach (REQUIRED_FIELDS as $field) {
        if (empty($fm[$field])) {
            $errors[] = "$path: missing required field '$field'";
        }
    }
    if (isset($fm['stack']) && !in_array($fm['stack'], ALLOWED_STACKS, true)) {
        $errors[] = "$path: invalid stack '{$fm['stack']}', must be one of " . implode(',', ALLOWED_STACKS);
    }
    if (isset($fm['type']) && !in_array($fm['type'], ALLOWED_TYPES, true)) {
        $errors[] = "$path: invalid type '{$fm['type']}', must be one of " . implode(',', ALLOWED_TYPES);
    }
    if (empty($fm['source']) && empty($fm['author'])) {
        $errors[] = "$path: must have either 'source' (curated) or 'author' (original)";
    }
    if (isset($fm['type']) && $fm['type'] === 'skill' && basename($path) !== 'SKILL.md') {
        $errors[] = "$path: skill files must be named SKILL.md";
    }
    return $errors;
}

function walk(string $target): array
{
    $errors = [];
    if (is_file($target)) {
        return str_ends_with($target, '.md') ? validateFile($target) : [];
    }
    $it = new RecursiveIteratorIterator(new RecursiveDirectoryIterator($target, FilesystemIterator::SKIP_DOTS));
    foreach ($it as $file) {
        if ($file->isFile() && $file->getExtension() === 'md') {
            $errors = array_merge($errors, validateFile($file->getPathname()));
        }
    }
    return $errors;
}

$target = $argv[1] ?? null;
if ($target === null) {
    fwrite(STDERR, "Usage: php lint-frontmatter.php <file-or-dir>\n");
    exit(2);
}

$errors = walk($target);
foreach ($errors as $e) {
    fwrite(STDERR, "ERROR: $e\n");
}
exit($errors === [] ? 0 : 1);
```

## Task 7: Index generator — implementation

**Files:**
- Create: `scripts/generate-index.php`

**Step 1: Write the generator**

Responsibilities:
1. Parse CLI flags: `--root=<dir>` (default: repo root), `--readme=<file>` (default: `README.md`), `--check` (do not write, exit 1 on drift).
2. Walk `<root>/php/` and `<root>/laravel/` for `.md` files (reuse the same parser as `lint-frontmatter.php` — copy-paste is fine for now, extract later if a third script shows up).
3. Build rows `{stack, type, name, description, paste-into}`.
4. Sort by (stack, type, name).
5. Render a markdown table.
6. Read the README, splice the table between `<!-- INDEX:START -->` and `<!-- INDEX:END -->`.
7. In normal mode: write the result back. In `--check` mode: compare against the current file and exit 1 if different.

Full implementation:

```php
<?php
// scripts/generate-index.php
declare(strict_types=1);

function parseFrontmatter(string $content): ?array
{
    if (!str_starts_with($content, "---\n")) {
        return null;
    }
    $end = strpos($content, "\n---", 4);
    if ($end === false) {
        return null;
    }
    $block = substr($content, 4, $end - 4);
    $out = [];
    foreach (explode("\n", $block) as $line) {
        if (!preg_match('/^([\w\-]+):\s*(.*)$/', $line, $m)) {
            continue;
        }
        $out[$m[1]] = trim($m[2], " \"'");
    }
    return $out;
}

$options = getopt('', ['root::', 'readme::', 'check']);
$root = $options['root'] ?? getcwd();
$readme = $options['readme'] ?? ($root . '/README.md');
$checkMode = isset($options['check']);

$rows = [];
foreach (['php', 'laravel'] as $stack) {
    $stackDir = "$root/$stack";
    if (!is_dir($stackDir)) {
        continue;
    }
    $it = new RecursiveIteratorIterator(new RecursiveDirectoryIterator($stackDir, FilesystemIterator::SKIP_DOTS));
    foreach ($it as $file) {
        if (!$file->isFile() || $file->getExtension() !== 'md') {
            continue;
        }
        $fm = parseFrontmatter(file_get_contents($file->getPathname()));
        if ($fm === null || empty($fm['name'])) {
            continue;
        }
        $rows[] = $fm;
    }
}

usort($rows, function ($a, $b) {
    return [$a['stack'], $a['type'], $a['name']] <=> [$b['stack'], $b['type'], $b['name']];
});

$table = "| Stack | Type | Name | Description | Paste into |\n";
$table .= "|-------|------|------|-------------|------------|\n";
foreach ($rows as $r) {
    $table .= sprintf(
        "| %s | %s | %s | %s | `%s` |\n",
        $r['stack'], $r['type'], $r['name'], $r['description'], $r['paste-into'] ?? ''
    );
}
$table = rtrim($table);

$current = file_get_contents($readme);
if ($current === false) {
    fwrite(STDERR, "Could not read $readme\n");
    exit(2);
}

$pattern = '/(<!-- INDEX:START -->)(.*)(<!-- INDEX:END -->)/s';
if (!preg_match($pattern, $current)) {
    fwrite(STDERR, "$readme is missing INDEX markers\n");
    exit(2);
}

$replacement = "$1\n$table\n$3";
$updated = preg_replace($pattern, $replacement, $current);

if ($checkMode) {
    if ($updated !== $current) {
        fwrite(STDERR, "README index is stale. Run: php scripts/generate-index.php\n");
        exit(1);
    }
    echo "README index is up to date.\n";
    exit(0);
}

file_put_contents($readme, $updated);
echo "Updated $readme\n";
exit(0);
```
